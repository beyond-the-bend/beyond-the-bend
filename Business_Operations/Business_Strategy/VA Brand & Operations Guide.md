## 2. BRAND COLOURS (Use These Exactly)

### Primary Colours
| Colour | Hex Code | Where to Use |
|--------|----------|-------------|
| Sage Green | #A8B5A2 | Backgrounds, section fills, soft accents |
| Soft Blush Pink | #E8C4C4 | Gentle highlights, decorative touches |
| Warm Sand | #E6D5C7 | Backgrounds, cards, warm sections |

### Accent Colours
| Colour | Hex Code | Where to Use |
|--------|----------|-------------|
| Deep Sage Green | #6B8063 | Buttons, strong accents, borders |
| Muted Rosewood | #B06568 | Sparingly, for warmth (not as main colour) |

### Text Colours
| Colour | Hex Code | Where to Use |
|--------|----------|-------------|
| Dark Olive | #3F4A3C | Main body text |
| Secondary Grey Green | #5A6658 | Subheadings, captions |
| Light Cream | #F7F4EF | Text on dark backgrounds |

### Rules
- **NEVER** use pure black (#000000)
- **NEVER** use harsh white (#FFFFFF) as a background (use #F7F4EF or #f7f7f5 instead)
- **NEVER** use neon, bright, or saturated colours
- When in doubt, go softer and more muted

## 3. FONTS (Use These Exactly)

| Font | Type | Where to Use |
|------|------|-------------|
| **Playfair Display** | Serif | Headlines, titles, thumbnails, quote graphics |
| **Lato** | Sans-serif | Body text, paragraphs, captions, emails, descriptions |

### Rules
- **NEVER** use script/cursive fonts
- **NEVER** use blocky or bold display fonts
- **NEVER** mix more than these two fonts
- Both fonts are available free on Google Fonts

## 4. BRAND VOICE & TONE

### The Voice Sounds Like:
- A calm, grounded, wise friend
- Warm but not overly sweet
- Thoughtful and human
- Quietly powerful
- Soulful without being "woo"

### The Voice Does NOT Sound Like:
- A fitness instructor
- A motivational speaker
- Corporate or professional jargon
- Trendy social media language
- Preachy or guru-like

### Writing Rules (CRITICAL)
1. **NO DASHES anywhere.** Not em dashes, not hyphens used as dashes. Restructure sentences instead.
2. **No jargon.** Use plain, warm, human language.
3. **No hype words** like "transform," "unleash," "unlock," "crush it," "level up."
4. **No one-line paragraphs** in public content. Write in natural flowing paragraphs.
5. **All public content ends with:**
   ```
   With warmth and gratitude,
   Laura ❤️
   ```

### Words We USE:
Grounded, Embodied, Somatic, Unhurried, Soulful, Authentic, Tender, Spacious, Permission, Safety, Gentle, Easeful, Quiet, Present, Rooted, Alive

### Words We AVOID:
Transform, Unleash, Unlock, Crush it, Level up, Boss babe, Hustle, Grind, Slay, Epic, Game changer, Manifest (overused), Tribe

## 5. IMAGERY STYLE

### Use:
- Natural light
- Soft shadows
- Linen or neutral clothing
- Easeful, natural movement
- Calm environments (studios, nature, soft interiors)
- Muted, earthy tones

### Avoid:
- Neon or bright colours
- Fitness/gym imagery
- Posed stock photos
- Before/after comparisons
- Anything that feels performative or intense

## 6. FILE NAMING CONVENTIONS

Use this format for all files:

| Type | Format | Example |
|------|--------|---------|
| Class recordings | YYYY-MM-DD_ClassName | 2025-02-10_GroundedFlow |
| Meditations | meditation_TopicName | meditation_QuietReturn |
| Graphics | graphic_Purpose_Date | graphic_IGpost_Feb15 |
| Newsletters | newsletter_TopicShort | newsletter_PsoasSoul |
| Course assets | course_CourseName_AssetType | course_QuietReturn_PDF |

### Rules
- Use underscores, not spaces
- Keep names short but descriptive
- No special characters

## 7. GOHIGHLEVEL (GHL) STANDARDS

### Membership Area
- Course/program titles: Use Playfair Display style (title case, elegant)
- Descriptions: Written in Lato style, warm and human
- Organize classes by date or by style (check with Laura)
- Every video needs a clear title and short description

### Pages
- Use brand colours only (refer to Section 2)
- Keep layouts clean and spacious (lots of breathing room)
- Use Playfair Display for headlines
- Use Lato for body text
- Always include soft, warm imagery

### Email Campaigns
- Use the HTML template Laura provides (or adapt from existing)
- Always include `{{unsubscribe_link}}`
- Always include `{{contact.first_name}}` for personalization
- End with Laura's signature

## 8. VIDEO WORKFLOW

### For Class Recordings (OBS)
1. Laura records the class via OBS
2. Diana adds the approved intro and outro
3. Export in high quality (1080p minimum)
4. Name following the file convention: `YYYY-MM-DD_ClassName`
5. Upload to GHL membership area under the correct category
6. Add a title and short description (1-2 sentences)
7. Notify Laura when complete

### Quality Check Before Upload:
- [ ] Intro plays correctly
- [ ] Outro plays correctly
- [ ] Audio is clear throughout
- [ ] Video is not cut off or cropped awkwardly
- [ ] Correct file name applied

## 9. COMMON MISTAKES TO AVOID

| Mistake | Correction |
|---------|------------|
| Using pure black text | Use Dark Olive #3F4A3C |
| Using a script/fancy font | Use Playfair Display or Lato only |
| Writing "transform your life" | Write something like "finding your way back to yourself" |
| Using a dash in writing | Restructure the sentence without a dash |
| Bright, saturated graphics | Muted, earthy, soft palette only |
| Leaving default white backgrounds | Use #F7F4EF or a soft warm tone |
| Generic stock photos | Natural, calm, real-feeling imagery |
