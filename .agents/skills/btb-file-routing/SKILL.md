# Beyond the Bend (BTB) File Organization Rules

When creating, saving, or modifying files for Laura (Beyond the Bend Yoga), you MUST save them into the correct directory within `C:\BTB`. 

Never save files loosely in the root `C:\BTB` folder or in your temporary workspace unless specifically instructed. 

## Folder Structure & Routing Rules

Always place newly generated content, code, or images into one of these 5 designated folders.

### 1. 🖼️ Brand Assets -> `C:\BTB\Brand_Assets`
*   **What goes here:** Any images, logos, vector files (SVG), generated images for newsletters, photos of Laura, GHL icons, and general branding materials.
*   **Subfolders:**
    *   `C:\BTB\Brand_Assets\Brand_Identity` (Documents defining Laura's voice, personality, target audience, and brand vibe)
*   **Actionable Note:** If you generate an image for a newsletter using `generate_image`, always move/save it into `C:\BTB\Brand_Assets` or `C:\BTB\Content\newsletters\`. Do not leave generated images stuck in the agent's brain/artifacts folder.

### 2. ✍️ Content -> `C:\BTB\Content`
*   **What goes here:** All written materials, HTML newsletters, Substack essays, blog posts, and text copy.
*   **Subfolders:**
    *   `C:\BTB\Content\newsletters` (HTML newsletters for GoHighLevel)
    *   `C:\BTB\Content\Substack` (Markdown essays for Substack)

### 3. 🧘‍♀️ Sanctuary -> `C:\BTB\Sanctuary`
*   **What goes here:** Files related to the paid membership area, recorded video classes, class plans, or Master Content for the Sanctuary.
*   **Subfolders:**
    *   `C:\BTB\Sanctuary\Sanctuary_Master_Content`
    *   `C:\BTB\Sanctuary\Finished - Ready for Membership`

### 4. 🌐 Tech and Web -> `C:\BTB\Tech_and_Web`
*   **What goes here:** Any code, scripts, JS automations, GoHighLevel setup scripts, video processing scripts, or website HTML/CSS code. 
*   **Subfolders include:**
    *   `C:\BTB\Tech_and_Web\website`
    *   `C:\BTB\Tech_and_Web\ghl-mcp` (GoHighLevel scripts)
    *   `C:\BTB\Tech_and_Web\Video-Processor`
    *   `C:\BTB\Tech_and_Web\Remotion-Video-Processor`

### 5. 🏢 Business Operations -> `C:\BTB\Business_Operations`
*   **What goes here:** Legal forms, PDFs, waivers, tax documents, financial information, business strategy notes, affiliate tracking, and legacy scripts that aren't actively used anymore.
*   **Subfolders include:**
    *   `C:\BTB\Business_Operations\Account_Information`
    *   `C:\BTB\Business_Operations\Affiliates`
    *   `C:\BTB\Business_Operations\Business_Strategy`
    *   `C:\BTB\Business_Operations\Financial_Sanctuary`
    *   `C:\BTB\Business_Operations\Legacy_Scripts_And_Code`

### Exceptions
*   `C:\BTB\Weekly_Checklist.html` stays in the root folder for easy access.
*   Hidden system folders (like `.git` and `.agents`) stay in the root folder.

---
**Core Golden Rule:** Every single time you generate or save a file, categorize it into one of these 5 main buckets. Do not pollute `C:\BTB` with loose files!
