Video Workflow Guide

Beyond the Bend Yoga

I. The Canva Designs

Use the YouTube Template (1920 x 1080) in Canva. This ensures your videos match perfectly for the fastest processing.

1. Create 4 Intros

Build one intro design for each class style: Restorative, Gentle Hatha, Somatic Hatha, and Somatic Vinyasa. Use Playfair Display for titles and Lato for instructions.

2. Use the Three-Scene Structure
  • Scene 1: The Sanctuary | [Class Name]
  • Scene 2: Instructions on gathering props (blankets, blocks, pillows).
  • Scene 3: Settle into a lying position. We will begin in a moment.
3. Export & Save

Save these as MP4 files in the folder:
C:\BTB\Video-Processor\assets_intro

II. The Descript Editing

This is where you refine the class recording before adding the branding.

1. The "Surgical" Edit

Use the "Remove Filler Words" tool to take out the "ums." Trim your 75-minute recording down to a beautiful, clean 60 minutes.

2. Export the Class

Export the clean class as a single MP4. Place it in the folder:
C:\BTB\Video-Processor\raw_footage

III. The Magic Stitch

This phase adds your branding to your class instantly, without any long render times.

1. Start the Tool

Open C:\BTB\Video-Processor. Right-click the file Process-Videos.ps1 and choose "Run with PowerShell."

2. Choose Your Intro

The screen will list the intros you made in Canva. Type the number for the class you are processing (1, 2, 3, or 4) and press Enter.

3. Completion

Wait about 60 seconds. Your branded video will appear in the folder:
C:\BTB\Video-Processor\finished_videos
