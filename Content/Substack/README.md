# How to Use These Substack Files

Each newsletter in this folder is a clean, paste-ready version 
formatted for Substack. Here's how to post one in under 5 minutes.

---

## How to Post on Substack

1. Go to substack.com and log in
2. Click the pencil/compose icon (top right) — "New Post"
3. Type your TITLE in the title field at the top
4. Type your SUBTITLE (optional — the essay title works well here)
5. Click in the body area and paste the content from the .md file below the title
6. Add your image: click the image icon in the toolbar, upload from C:\BTB\newsletters\
7. Preview it — click "Preview" to see how it looks
8. When ready: click "Publish" → choose "Send to everyone" OR "Save as draft"

---

## What's Included in Each File

Each Substack version:
✅ Personal intro (edited to remove the GHL merge tag)

✅ Practice of the Week
✅ Closing thought
✅ Brief "From the Studio" mention at end (if applicable) — no full class table
❌ Class schedule tables removed (too promotional for Substack)
❌ Affiliate links removed
❌ GHL unsubscribe links removed (Substack handles this automatically)
