HOW TO USE THE VIDEO PROCESSOR
================================

This tool adds your Intro and Outro to your class videos in seconds, without any losing quality or waiting hours for rendering.

1. DIRECTORY GUIDE:
   - assets_intro: Put ONE intro video file here.
   - assets_outro: Put ONE outro video file here.
   - raw_footage: Put your 1-hour class recordings here (you can put multiple!).
   - finished_videos: Your final versions will appear here.

2. HOW TO RUN:
   - Right-click the file "Process-Videos.ps1"
   - Choose "Run with PowerShell"

3. TIPS FOR SUCCESS:
   - For the fastest speed, make sure your Intro, Outro, and Main Video are all the same size (e.g., all 1080p).
   - If they are different sizes, the video might look stretched or fail. If that happens, let AntiGravity know and I can adjust the script to "Smart Render" them for you.
