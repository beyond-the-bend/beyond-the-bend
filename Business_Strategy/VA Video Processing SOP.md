## STEP 1: Find the Project in Descript

1. Open Descript (you should already have shared workspace access)
2. Look for projects Laura has marked as ready
3. Open the project

**If you cannot find a project or something looks wrong, message Laura. Do not edit a project that hasn't been trimmed yet.**

---

## STEP 2: Add the Branded Intro

The intro is a **still image** (not a video clip) placed at the very beginning.

### What the intro includes:
- Beyond the Bend logo
- Laura's name
- Welcome text (asking students to gather props, settle in, option to put on their own music)
- Background music (gentle, short, fades before the class begins)

### How to add it in Descript:
1. Place your cursor at the very beginning of the timeline
2. Insert the intro image file (drag it in or use Insert > Media)
3. Set the image duration to approximately **15-20 seconds** (enough time to read the text)
4. Add the background music track underneath the intro image only
5. Fade the music out before the class audio begins
6. Make sure there is no gap or overlap between the intro and the class

### Rules:
- Music plays ONLY during the intro, never during the class itself
- The transition from intro to class should feel smooth and calm
- Use brand colours and fonts on the intro image (see Brand Guide)

---

## STEP 3: Add the Branded Outro

The outro is a **still image** placed at the very end.

### What the outro includes:
- A gentle closing message (thank you / goodbye)
- Beyond the Bend logo
- Optional: website URL or social handle

### How to add it in Descript:
1. Place your cursor at the very end of the timeline (after Laura's class ends)
2. Insert the outro image file
3. Set the image duration to approximately **8-10 seconds**
4. Optional: add soft music underneath the outro if Laura approves
5. Make sure the transition from class to outro feels gentle (no abrupt cut)

---

## STEP 4: Quality Check (Before Export)

Run through this checklist before exporting:

- [ ] Intro image plays correctly at the start
- [ ] Intro text is readable and uses brand fonts/colours
- [ ] Intro music plays and fades out before class begins
- [ ] No music plays during the actual class
- [ ] No gaps or awkward jumps between intro and class
- [ ] Class audio is clear throughout
- [ ] Outro plays correctly at the end
- [ ] No abrupt cuts at any point
- [ ] Total video length is approximately 60 minutes (give or take)

---

## STEP 5: Export from Descript

1. Go to File > Export (or the Export button)
2. **Resolution:** 1080p (1920x1080)
3. **Format:** MP4
4. **Quality:** High
5. Save/download the exported file
6. **Name the file** using this convention:
   ```
   YYYY-MM-DD_ClassName
   ```
   Example: `2025-01-15_GroundedFlow`

---

## STEP 6: Upload to GoHighLevel Membership

### How to add a video to the membership area in GHL:

1. Log into GoHighLevel
2. Go to **Memberships** (in the left sidebar, under "Sites")
3. Find **The Sanctuary** membership (or the membership Laura has set up)
4. Open the **Course/Category** where this video belongs
   - If unsure which category, check with Laura
5. Click **Add Lesson** or **Add New Lesson**
6. **Lesson Title:** Use a clear, warm name. Examples:
   - "Grounded Flow: February 10"
   - "Restorative Stillness: January 22"
   - "Gentle Movement for Anxious Days"
7. **Upload the video:** Use the video upload option in the lesson editor
   - GHL may let you upload directly or paste a video URL
   - If direct upload: upload the MP4 from Step 5
   - If it requires a URL: upload the video to Google Drive first, get a shareable link, and paste it
8. **Add a short description** (1-2 sentences). Example:
   > "A gentle, grounded flow class to help you come back to yourself. Props: a block and a bolster if you have one."
9. **Save** the lesson
10. **Preview** the lesson to make sure the video plays correctly

### Troubleshooting:
- If the video won't upload (file too large), try compressing it first or contact Laura
- If you can't find the membership area, go to Sites > Memberships in the sidebar
- If there is no category for this class type yet, message Laura before creating one
